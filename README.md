## 💻 commit 규칙

- Commit
    - Commit 규칙을 통해 각 branch 별 commit 사항을 관리한다.

    <br>

    - commit message
        - 형식

            ```markdown
            <type> (<scope>): <explain>          
            ```

            <br>

        - type
            - feat : 새로운 추가 파일에 대한 커밋
            - update : 수정에 대한 커밋
            - img : 이미지 파일 추가에 대한 커밋


        <br>
        
        ex) commit message example
        
          `feat (basicdesign.md) : 기본설계 md 파일 생성`
        
          `update(basicdesign.md) : 기본설계 A내용 수정`
        
          `img(basicdesign.md) : img파일 추가`
